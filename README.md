# FitTracker

A simple fitness workout tracker web app built with HTML, CSS, and JavaScript.  
Users can add, delete, mark as complete, and save workouts.

## Features
- Add workout (exercise name, sets, reps)
- Delete workout
- Mark as complete / undo
- Persist workouts with localStorage
- Clean dark theme UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository.
2. Open `index.html` in a web browser.
3. Start adding workouts!
